# 学生就业信息管理系统

## 项目简介
学生就业信息管理系统是针对大学毕业生就业情况进行统计和管理的系统，通过此系统可以方便对毕业生的就业情况进行管理。模块上分：就业信息、就业统计、用户管理、登录/登出三个主要模块。
